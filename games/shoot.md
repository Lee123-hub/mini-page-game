一、核心概念
这是一个横向单轴移动的自动射击游戏：

玩家只能左右移动
玩家始终位于中间偏左区域，自动朝右射击
左侧刷新宝箱，玩家可移动靠近并射击击破
右侧刷新怪物，怪物持续进攻
玩家通过选择打哪些宝箱获得不同强化，逐步形成 build

核心乐趣不是“手动瞄准”，而是：

走位
战斗中做选择
构筑成长
不同流派成型后的爽感


二、基础玩法循环
每局流程：
1. 战斗波次开始

右侧刷怪
左侧定期出现 1~3 个不同类型宝箱
玩家需要在“清怪”和“打箱子拿强化”之间权衡

2. 决策
玩家只有左右移动，所以决策非常纯粹：

往右：更贴近怪物，优先压制战线
往左：更安全，更容易击破宝箱拿 build 强化

这会形成天然 tension：

现在去拿宝箱，可能漏怪
现在先守线，可能错过关键 build 组件

3. Build 成型
随着击破不同宝箱，子弹效果叠加：

攻击形态变化
异常状态变化
输出节奏变化
生存能力变化

4. 波次结算 / Boss / 更高难度

每 5 波一个精英
每 10 波一个 Boss
后期通过怪物抗性和词缀逼迫玩家做真正的 build 选择，而不是无脑全拿


三、场景与空间设计
虽然只能左右移动，但空间要有博弈感。
场景布局

左侧：宝箱带
宝箱在左边 20%~30% 区域刷新
不同宝箱有不同耐久和危险度
中部：玩家活动主区域
玩家大多数时间在这里游走
右侧：怪物出生区
怪物从右边不断向左推进

为什么左箱右怪好玩
因为它天然制造了一个选择：

你去左边打箱子，就等于暂时放松右侧防线
你一直顶右边，就会错失 build 成型机会

这比纯随机三选一升级更有操作感。

四、玩家基础属性
玩家初始拥有：

自动射击
基础直线子弹
固定射程
基础攻速
基础伤害
基础生命值
受击后短暂无敌

可成长核心维度

伤害
射速
弹道数量
穿透/连锁/范围
元素效果
暴击/处决
生存能力
资源获取效率


五、宝箱系统设计
这是核心。
宝箱刷新规则
每隔固定时间，在左侧刷新宝箱：

普通宝箱：基础强化
稀有宝箱：流派核心件
诅咒宝箱：高收益高风险
专属宝箱：当你已有某流派倾向时更容易出现

宝箱击破逻辑

玩家必须主动靠近左侧并射击击破
宝箱有耐久
某些宝箱会限时消失
某些宝箱击破后会产生副作用

这样宝箱不是“送奖励”，而是“战斗中的决策目标”。

六、宝箱种类设计
下面按功能分层。
1. 基础属性箱
适合所有流派。
迅捷箱

射速 +15%
小概率额外 +5% 移速
用于快攻流

重装箱

子弹伤害 +20%
射速 -5%
用于重炮流

扩容箱

弹道数 +1
单发伤害 -10%
用于散射/特效触发流

校准箱

暴击率 +8%
暴击伤害 +20%
用于高爆发流


2. 元素效果箱
毒液箱

子弹附带中毒
中毒持续掉血，可叠层
适合持续伤害 build

火焰箱

子弹附带点燃
点燃可小范围溅射
适合清杂与范围压制

激光箱

子弹变为高穿透激光束
命中路径上多个敌人
适合直线压制 build

电弧箱

命中后可连锁到附近敌人
适合群体扩散流

冰霜箱

子弹附带减速
叠满后可冻结短暂定身
偏控制流


3. 机制变化箱
分裂箱

子弹命中后分裂为 2~3 枚小弹
适合弹幕型流派

穿透箱

子弹穿透 +1/+2
适合走廊清线

反弹箱

子弹命中边界或敌人后反弹
对密集怪群很好用

追踪箱

子弹轻微追踪敌人
提高稳定性

充能箱

连续射击后进入过热充能态
充能结束后短时间高爆发
适合节奏型 build


4. 风险收益箱
诅咒箱

攻击大幅提升
但玩家最大生命下降
高风险玻璃炮

赌徒箱

随机获得 2~3 项大强化中的一项
也可能附带小负面
增加局内戏剧性

献祭箱

扣当前生命 20%
换一个高阶专属强化
适合后期搏成型


七、至少三种明确 build 流派
我给你 5 套，你至少能先做 3 套上线。

Build 1：毒爆扩散流
核心体验
前期靠中毒磨血，中后期通过叠毒、引爆、扩散形成“越打越疼”的滚雪球。
核心组件

毒液箱：普攻附毒
迅捷箱：提高叠毒效率
扩容箱：更多弹道触发更多中毒
分裂箱：命中后扩散更多毒源
腐蚀箱（专属）
中毒敌人受到额外伤害
毒爆箱（专属）
敌人死亡时引爆毒层，对周围造成范围毒伤

成型逻辑

前期：叠毒为主，清怪一般
中期：多弹道+射速起来后，毒层叠加速度明显提升
后期：靠“毒爆传播”形成连锁清屏

优势

持续输出强
清密集怪很爽
对高血量敌人表现好

劣势

前期爆发不足
对瞬时击杀需求高的 Boss 战略弱


Build 2：激光贯穿流
核心体验
把普通子弹改造成高频穿透激光，站住中轴后一路横扫。
核心组件

激光箱：攻击形态切换为激光
穿透箱：进一步增加贯穿目标数
重装箱：提升单次命中伤害
校准箱：提高暴击与激光爆发
聚焦镜箱（专属）
激光变细但伤害显著提升
棱镜箱（专属）
激光命中后折射一次
过载箱（专属）
连续命中同一目标伤害递增

成型逻辑

前期：拿到激光箱后体验立刻变化
中期：配穿透和伤害箱开始稳定压线
后期：聚焦/过载成型后对精英和 Boss 极强

优势

直线清线最强
对前排怪和 Boss 输出稳定
手感直接，反馈爽

劣势

对散开的多目标处理不如扩散流
比较依赖关键核心箱出现


Build 3：火焰霰弹流
核心体验
靠多弹道、近距离高射速、点燃溅射打出“贴脸烧穿”的爽感。
核心组件

火焰箱：附加点燃
扩容箱：增加弹道数
迅捷箱：提高喷发频率
近爆箱（专属）
距离越近伤害越高
爆燃箱（专属）
被点燃敌人死亡后爆炸
炽流箱（专属）
连续攻击时锥形范围变宽

成型逻辑

前期：像小霰弹枪，清小怪舒服
中期：通过点燃爆炸扩散清场
后期：近距离站位风险高，但回报极强

优势

清杂速度快
贴脸输出爽
视觉反馈很强

劣势

站位危险
面对远程精英压力大


Build 4：电弧连锁流
核心体验
单发命中后不断跳电，越多敌人越强。
核心组件

电弧箱
迅捷箱
扩容箱
连锁箱（专属）：连锁次数 +1/+2
超载箱（专属）：同一片区域跳电次数越多，伤害越高
电容箱（专属）：短时间未命中后下次攻击释放大电弧

特点

群怪环境极强
对单 Boss 一般
画面表现非常容易做出爽感


Build 5：暴击重炮流
核心体验
射速不快，但每一发都很重，追求暴击、斩杀和一枪穿透。
核心组件

重装箱
校准箱
穿透箱
处决箱（专属）：低血敌人额外斩杀
破甲箱（专属）：连续命中同一敌人降低其抗性
蓄势箱（专属）：短暂停火后下一发超高伤

特点

打精英/Boss 很舒服
清小怪依赖穿透和分裂
很适合偏数值爽感用户


八、建议的三大首发流派
如果你想先做一个最小可玩版本，我建议首发就做这三套：
1. 毒爆扩散流
原因：

容易做“叠层—引爆—传播”的成长感
有明显 build 认知
和普通直伤区分大

2. 激光贯穿流
原因：

非常直观
手感变化大
很适合作为“拿到核心箱立即爽”的流派

3. 火焰霰弹流
原因：

高风险高回报
近身打法和另外两派差异明显
演出表现好做

这三套已经足够形成：

持续 DOT
直线爆发
近战散射
三种完全不同的体验。


九、怪物设计建议
为了让 build 真有意义，怪物不能只是血量不同。
基础怪

直线推进
血少量多
检验清杂能力

护甲怪

减少低伤多段效果
克制小毒小火
逼玩家补破甲或高伤

快速怪

移速快
克制慢热 build
逼迫玩家守线

远程怪

远距离发射弹丸
压缩玩家左右走位空间

自爆怪

靠近后爆炸
对近战火焰流形成压力

精英怪

带词缀，如：
抗毒
抗火
护盾
召唤
分裂
冲锋

这样玩家不能无脑拿同一套，而需要适配。

十、宝箱与 build 的博弈机制
为了让“选择打哪个箱子”更有意思，可以做这些规则：
1. 同屏只存在有限宝箱
比如每次 2 选 1 或 3 选 1：

毒液箱
激光箱
射速箱

你去打哪个，就意味着你放弃另一个。
2. 倾向性掉落
如果玩家已经拿了 2 个毒系组件：

后续更高概率刷毒系专属箱
但也保留跨流派转型可能

3. 宝箱有限时
宝箱存在 8~12 秒

不打就消失
强迫玩家做战场决策

4. 宝箱耐久不同

稀有箱更难打碎
高价值箱往往需要更久停留在左侧

这会非常刺激。

十一、局外成长建议
如果你想做长线体验，可以加轻 Roguelite：

初始生命提升
初始攻速提升
稀有箱出现率提升

但建议别一开始做太重，不然会冲淡局内 build 的乐趣。

十二、数值节奏建议
前期 1~3 波

很容易拿到基础属性箱
让玩家快速理解“左打箱、右打怪”

中期 4~8 波

开始出现元素箱和机制箱
build 方向逐渐明确

后期 9 波以后

出专属箱和诅咒箱
怪物词缀出现
要求玩家有成型流派

Boss 战
Boss 设计成检验不同流派：

大体型直线 Boss：激光流强
召小怪 Boss：毒爆/火焰流强
高机动 Boss：追踪/电弧流更舒服

十四、我建议你重点强调的卖点
如果是对外讲这个游戏，可以这样总结：
卖点 1：极简操作

玩家只需左右移动
自动射击，低门槛

卖点 2：高密度决策

左侧打箱构筑，右侧守线求生
每次移动都在做选择

卖点 3：强 build 爽感

毒、火、激光、多弹道、分裂、穿透等效果叠加
每局都能打出不同流派

卖点 4：战斗中构筑

不是菜单里选天赋
而是在实时战斗里抢宝箱做 build
